RTRA program generator

Before preparing "code…"
  • Step 1: For the statistic you are interested in, identify the data source or survey. You can get the "same" statistic from different sources. For example, you can % employed from the Census, APS and LFS.
  • Step 2: Get the data dictionary (also called code book). You can request it from Mohan Kumar (mohan.kumar@canada.ca) or Henry Robertson (henry.robertson@canada.ca) or download it using FileZilla using the instructions provided here: https://www.statcan.gc.ca/eng/rtra/ftp
  • Step 3: familiarize yourself with the data dictionary. This can be daunting since there are thousands of variables. But, you can start by finding variables of interest by searching the PDF or Word document (Ctrl + F).
  • Step 4: Get the document “rtrap.dox” (RTRA Parameters) from https://www.statcan.gc.ca/eng/rtra/rtrap
  • Step 5: In the document RTRA Parameters, find the data source or survey. For example, if you are interested in the 2012 Aboriginal Peoples Survey (APS), go to “APS 2012.” Here, get the:
    1. “Dataset Name”
    2. “Weight Name”
Note the deleted variables. These are variables you will see in the data dictionary but wouldn’t be able to use them.
